# LS - Assignment

## Pre-requisites

- Node.js 18.17 or later

## How to setup and use

Install packages:

```bash
npm install
```

Run in production mode:

```bash
npm run build
npm start
```

Or, run in development mode:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the app.
